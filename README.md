A full-stack Job Tracker application that allows users to manage and monitor their job applications efficiently. The application provides features to add, view, update, and delete job records through a responsive user interface.

The backend is built using Spring Boot with RESTful APIs and JPA for database interaction, while the frontend is developed using React (Vite). The application uses MySQL as the database and integrates frontend and backend seamlessly using API calls.

This project demonstrates end-to-end full-stack development, including API design, database integration, and dynamic UI rendering.
